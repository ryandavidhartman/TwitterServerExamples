# Quickstart

## Server

sbt 'runMain Server'
test with curl -D - localhost:8080

## Clients

sbt 'runMain Client'
Hard coded to make an HTTP request to www.scala-lang.org

## Proxy

sbt 'runMain Proxy'
proxies twitter.com on localhost:8080
